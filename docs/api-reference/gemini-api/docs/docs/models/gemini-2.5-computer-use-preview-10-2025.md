# Gemini 2.5 Computer Use model

<br />

> [!NOTE]
> **Note:** The [Gemini 3](https://ai.google.dev/gemini-api/docs/models#gemini-3) Pro and Flash models support built-in computer use, without the need for a separate model.

An agentic capability model designed for direct interface interaction, allowing
Gemini to perceive and navigate digital environments. Gemini 2.5 Computer Use is
best for browser automation, multi-step software tasks, and UI testing that
requires visual reasoning and precise cursor control.

Visit the [Computer use](https://ai.google.dev/gemini-api/docs/computer-use) page for full coverage
of features and capabilities.
