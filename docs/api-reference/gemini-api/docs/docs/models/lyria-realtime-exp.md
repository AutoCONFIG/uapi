# Lyria RealTime experimental

<br />

An experimental engine for high-fidelity musical synthesis, offering innovative
capabilities for generating and transforming audio content. Lyria RealTime
is best for AI-assisted songwriting, unique instrumental generation, and
creative audio workflows that require deep control over melody and rhythm, with
no vocals.
[Try Lyria RealTime](https://deepmind.google/models/lyria/lyria-realtime/)

## Documentation

Visit the [Realtime music generation](https://ai.google.dev/gemini-api/docs/realtime-music-generation)
page for full coverage of features and capabilities.
