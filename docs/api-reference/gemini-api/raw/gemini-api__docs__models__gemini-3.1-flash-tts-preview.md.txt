# Gemini 3.1 Flash TTS (Text-to-Speech) Preview

<br />

The Gemini 3.1 Flash TTS Preview model provides powerful, low-latency speech
generation with natural outputs, steerable prompts, and new expressive audio
tags for precise narration control.
[Try in Google AI Studio](https://aistudio.google.com?model=gemini-3.1-flash-tts-preview)

## Documentation

The Gemini 3.1 Flash TTS Preview model introduces expressive [audio tags](https://ai.google.dev/gemini-api/docs/speech-generation#transcript-tags)
for controlling narration, as well as overall improvements to naturalness,
controllability, and multilinguality.

Visit the [Text-to-Speech](https://ai.google.dev/gemini-api/docs/speech-generation) guide for full
coverage of features and capabilities.
