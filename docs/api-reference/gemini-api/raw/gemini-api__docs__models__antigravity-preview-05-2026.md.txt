# Antigravity preview

<br />

A powerful general-purpose managed agent designed for autonomous, multi-step
workflows. This model plans, reasons, runs code, manages files, and searches the
web inside a secure, isolated Linux sandbox hosted by Google.

## Documentation

Visit the [Antigravity Agent](https://ai.google.dev/gemini-api/docs/antigravity-agent) page for full
coverage of features, capabilities, and code samples.
