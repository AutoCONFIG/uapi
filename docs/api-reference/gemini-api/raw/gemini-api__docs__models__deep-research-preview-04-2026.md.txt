# Deep Research preview

<br />

A powerful agentic researcher designed for autonomous, multi-step investigations
that synthesize complex information into comprehensive, cited reports. This
model supports collaborative planning, visualization, MCP servers, and File
Search. It comes in two versions: Deep Research for speed and efficiency, and
Deep Research Max for maximum comprehensiveness.

## Documentation

Visit the [Deep Research](https://ai.google.dev/gemini-api/docs/deep-research) page for full coverage
of features and capabilities.
